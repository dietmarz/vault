= docker

In bridge mode, ist der linke port nur für localhost außerhalb des Dockernetzwerkes, innerhalb des netzwerkes gelten die service namen als DNS namen und die RECHTEN! Portnummmern!
Somit ist es nicht möglich zwei fix mit port 22 gelieferten images in einem compose laufen zu lassen.


== Settings
* Disable proxy settings
* Use openshift credentials with
  ** external.dietmar.zabel
  ** and Lastpass oepenshift passwort starting with U?R
